Templates: examples

Base template

To keep your template files DRY, it's helpful to have a base template that includes any content that's common to every page, like the header, main menu, and footer. Thanks to Jinja's template inheritance, this base template can include "blocks" that can be populated by more specific templates.

Here's a simple base template with blocks for:

  • title
  • description
  • body
<!doctype html>
<html lang="en-CA">
<head>
    <title>{% block title %}{% endblock %} ¬∑ {{ sourceOrganization.name }}</title>
    <meta charset="utf-8">
    <meta name="viewport"
        content="width=device-width, initial-scale=1">
    <meta name="description"
        content="{% block description %}{% endblock %}">
    <link rel="canonical"
        href="{{ url }}">
    {{ open_graph_tags|safe }}
    {{ schema_data_tag|safe }}
</head>
<body>
    <main>{% block body %}{% endblock %}</main>
</body>
</html>

Basic page template

Here is a basic WebPage.html template. The title, description, and body blocks will populate the base template above.

{% extends "base.html" %}

{% block title %}{{ name }}{% endblock %}
{% block description %}{{ abstract }}{% endblock %}

{% block body %}
<article>
  {{ body|safe }}
</article>
{% endblock %}

Blog index with a list of posts

This is a simple Blog template (Blog.html) with a list of posts sorted newest to oldest:

{% extends "base.html" %}

{% block title %}{{ name }}{% endblock %}
{% block description %}{{ abstract }}{% endblock %}

{% block body %}
{{ body|safe }}

{% if sibling_pages %}
<h2>Posts</h2>

{% for page in sibling_pages|sort(reverse=true, attribute="datePublished") %}
<div>
  <h3><a href="{{ page.url }}">{{ page.headline }}</a></h3>
  <div>
    üóì <time datetime="{{ page.datePublished }}">{{ page.datePublished }}</time> ¬∑
    üè∑ {% for keyword in page.keywords.split(',') %}{{ keyword }}{% if not loop.last %}, {% endif %}{% endfor %}

{% endblock %}
