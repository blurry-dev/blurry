# Commands: `clean`

## Description

`clean` removes the build folder and all of its contents.

The build folder is specified by the `build_directory_name` [setting](./../configuration/settings.md), and defaults to `./dist/`.

## Usage

Example usage:

```shell
$ blurry clean
```
